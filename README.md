# Insta_Firebase
Instagram Like with implementation of Firebase   
You can:
- Add user
- Add post
- Comment post
- Like post
- Follow users
- and more....

![Insta Firebase](https://user-images.githubusercontent.com/49925096/82474485-6a427400-9acb-11ea-9d69-540cdbe88a75.gif)
